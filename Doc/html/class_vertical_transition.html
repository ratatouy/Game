Detailed Description

Vertical Transition to a neighboring scene.

A Vertical Transition is essentially a line caracterized by two points on the same x axis That activates if the player is between or past those two points. The Transition can either go up or go down. Usually the transition is placed on the top or bottom border of the scene.
