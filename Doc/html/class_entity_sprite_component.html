Detailed Description

Component to manage the sprites of an entity.

This stores a map of named sprites and their data (the sprite, the local transformable and the texture). Only one EntitySpriteComponent is attached to each entity.

Created in order to support multiple sprites attached to a single entity.

Parameters
nameName of the sprite

Implementation :

  • Load the texture
  • Create the sprite
