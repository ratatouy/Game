Detailed Description

Component to manage the sprites of an entity.

Created in order to support multiple sprites attached to a single entity.

void EntitySpriteComponent::AddSprite ( std::string  name,
const char *  filepath 
)

Adds a sprite to the entity

Parameters
nameName of the sprite
filepathPath to the texture

Implementation :

  • Load the texture
  • Create the sprite
  • Set the sprite's position, rotation and scale to the entity's position, rotation and scale
  • Add the sprite's local Transformable to the "localSpriteTransformables_" vector
  • Add the sprite and texture to the "sprites_" and "textures_" vectors

void EntitySpriteComponent::update ( )

Updates all of the sprites

Implementation : By summing the entity's position, rotation & origin with the sprite's local position and rotation , Multiplying the entity's scale with the sprite's local scale

Note
not optimal, updates everything, every frame
