Detailed Description

An Entity is an object that has an update and a render method.

This class is abstract. The sf::Transformable and EntitySpriteComponent are mandatory and public.

Update the entity.

"Update" might be a bit misleading as it doesn't fully update the entity. What I mean is it only updates what is possible to update without looking at other entities. Inter-entity update is managed in the scenes :

See also
Scene::update()
Note
Order of Updating should be :
  • independant update (like move or process inputs)
  • collisions (check & process)
  • update EntitySprite Component (
See also
EntitySpriteComponent::update)
