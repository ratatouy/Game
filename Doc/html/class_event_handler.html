Detailed Description

Manages the event queue.

You have to move the ownership of the events to the EventHandler with std::move This can only :

  • Add an event to the end of the queue
  • Return the current event
  • Process the current event
Warning
Do not assign the same event to the queue twice, it will cause memory leaks
