\doxysubsection{Detailed Description}
Manages transitions from the border of a screen to a neighboring screen. 

This class is the base class for \mbox{\hyperlink{class_vertical_transition}{Vertical\+Transition}} and \mbox{\hyperlink{class_horizontal_transition}{Horizontal\+Transition}}. And is not intended to be used directly.
