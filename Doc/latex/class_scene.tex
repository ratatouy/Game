\doxysubsection{Detailed Description}
Base class for all Scenes. 

The \mbox{\hyperlink{class_scene}{Scene}} is the mastermind behind everything, it links every elements together. It has references to the \mbox{\hyperlink{class_render_engine}{Render\+Engine}}, the Physics\+Engine, the \mbox{\hyperlink{class_event_handler}{Event\+Handler}} and the entities.

\begin{DoxyNote}{Note}
missing elements \+: \begin{DoxyItemize}
\item Transitions \item Solid Tiles \item Decorations (bg assets/...) \end{DoxyItemize}

\end{DoxyNote}
