\doxysubsection{Detailed Description}
Base class for all Scenes. 

All of the game is just a collection of scenes. Each scene is a box, with a collection of \+: \begin{DoxyItemize}
\item transitions \item entities\end{DoxyItemize}
\begin{DoxyNote}{Note}
missing elements \+: \begin{DoxyItemize}
\item Transitions \item Solid Tiles \item Decorations (bg assets/...) \end{DoxyItemize}

\end{DoxyNote}
