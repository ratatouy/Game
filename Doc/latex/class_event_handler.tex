\doxysubsection{Detailed Description}
Manages the event queue. 

It uses a queue of \mbox{\hyperlink{class_event}{Event}} pointers

\begin{DoxyNote}{Note}
Methods \+:
\begin{DoxyItemize}
\item Add any event to the end of the queue
\item Return the current event
\item Process the current event
\end{DoxyItemize}
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Do not assign the same event to the queue twice, it will cause memory leaks 
\end{DoxyWarning}
