\doxysubsection{Detailed Description}
Manages the event queue. 

You have to move the ownership of the events to the \mbox{\hyperlink{class_event_handler}{Event\+Handler}} with std\+::move This can only \+: \begin{DoxyItemize}
\item Add an event to the end of the queue \item Return the current event \item Process the current event\end{DoxyItemize}
\begin{DoxyWarning}{Warning}
Do not assign the same event to the queue twice, it will cause memory leaks 
\end{DoxyWarning}
