\doxysubsection{Detailed Description}
Main class of the game. 

The game is a collection of scenes, with only one active at a time. It also is a singleton. It\textquotesingle{}s role is to bring together the \mbox{\hyperlink{class_render_engine}{Render\+Engine}}, \mbox{\hyperlink{class_event_handler}{Event\+Handler}} and the active \mbox{\hyperlink{class_scene}{Scene}}.
