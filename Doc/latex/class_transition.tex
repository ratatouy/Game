\doxysubsection{Detailed Description}
Object to manage transitions between Scenes. 

It\textquotesingle{}s an abstract class. A transition can be of any kind, it just needs a target \mbox{\hyperlink{class_scene}{Scene}} and a way to check if it should be activated.
